# DirParse

# Quickly-use
```
git clone https://github.com/bilibiliganb/dirparse.git

cd dirparse

make

cp /path/dir.txt ./

./dirparse 

You will get test.db.


Then execute the search command and append the parameters. You can search the contents of dir.txt.

./searchall log
C:\Windows\system32\logs
C:\Windows\system32\etc\C:\Windows\system32\logs
C:\Windows\system32\logs\log1.txt
C:\Windows\system32\logs\log2.txt



```
